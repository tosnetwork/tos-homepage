AGIW (Proof-of-Intelligent-Work) is TOS's settlement primitive that turns agent work into verifiable receipts—not a consensus mechanism.

How it works:

  • Agent completes task (e.g., generates image, answers query)
  • Output + proof (TEE attestation or quorum signature) → work receipt
  • Receipt submitted on-chain
  • Random spot-checks verify correctness
  • If valid → agent paid; if fraudulent → slashed

This decouples work verification from consensus (GhostDAG), allowing high-throughput settlement of AI tasks without bloating the main chain.

TOS balances privacy with verifiable compliance through:

  • Safety Oracle: At the Account Abstraction layer, checks work receipts against policies (e.g., no hate speech generation) without seeing raw data
  • Policy Wallets: Agents can prove "I comply with policy X" to third parties using zero-knowledge proofs
  • Selective Disclosure: DID credentials reveal only necessary attributes (e.g., "reputation >90") without exposing full history

This enables regulatory compliance (e.g., GDPR, AI safety rules) without sacrificing decentralization or agent autonomy.

TOS advances blockchains from "general computation" to "general economic agency":

  • Other chains: Execute code (smart contracts), settle value transfers
  • TOS: Settles work, anchors identity, governs compliance—purpose-built for agents earning autonomously

Technical differentiators:

  • 10,000 TPS + 1-second blocks via GhostDAG (BlockDAG architecture)
  • Native DID/AGIW/CC-EC primitives (not bolt-on smart contracts)
  • Pure PoW security (no staking centralization)
  • Agents as first-class citizens (not afterthoughts)
